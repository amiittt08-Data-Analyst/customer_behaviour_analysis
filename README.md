# End-to-End Data Analytics Project

## 📌 Project Overview
This project demonstrates a complete end-to-end Data Analytics workflow using Python, SQL, and Power BI. The objective of the project is to analyze customer shopping behavior and generate business insights that help improve decision-making.

The project covers:
- Data loading and cleaning
- Exploratory Data Analysis (EDA)
- SQL-based business analysis
- Interactive Power BI dashboard creation
- Report generation
- Presentation creation using Gamma AI

This project simulates a real-world company-level analytics workflow followed by data analysts in organizations.

---

## 📂 Dataset
The dataset contains retail customer shopping and transaction data, including:
- Customer demographics
- Product categories
- Purchase history
- Sales transactions
- Revenue information

The dataset was used to analyze customer behavior, sales trends, and business performance.

## 📊 Project Workflow

### 1. Data Loading
- Imported raw dataset into Python using Pandas
- Checked dataset structure and data types

### 2. Data Cleaning
- Handled missing values
- Removed duplicate records
- Fixed inconsistent data entries
- Converted columns into proper formats

### 3. Exploratory Data Analysis (EDA)
Performed analysis to identify:
- Customer purchasing trends
- Top-selling categories
- Revenue patterns
- Customer demographics insights

### 4. SQL Business Analysis
Executed SQL queries to solve business problems such as:
- Total sales analysis
- Monthly revenue trends
- Customer segmentation
- Best-performing product categories
- High-value customers

### 5. Dashboard Creation
Built an interactive Power BI dashboard with:
- KPI cards
- Sales trend charts
- Customer insights
- Filters and slicers
- Business performance metrics

### 6. Reporting & Presentation
- Created a detailed project report
- Designed a presentation deck using Gamma AI
- Documented findings and recommendations

## ▶️ How to Run the Project

### Step 1: Clone the Repository
```bash
git clone <your-github-repository-link>
```

### Step 2: Install Required Libraries
```bash
pip install pandas numpy matplotlib
```

### Step 3: Run Jupyter Notebook
Open and run:
```bash
Python_EDA.ipynb
```

### Step 4: Run SQL Queries
Execute the SQL queries in:
```bash
SQL_Queries.sql
```
using PostgreSQL, MySQL, or SQL Server.

### Step 5: Open Power BI Dashboard
Open:
```bash
PowerBI_Dashboard.pbix
```
in Power BI Desktop.
